# Task Manager DevSecOps

Aplicação de gerenciamento de tarefas pessoais, desenvolvida como atividade prática da disciplina **DevOps & DevSecOps** — UniSENAI.

## Objetivo

Aplicar os conceitos de SDLC, Secure SDLC, DevOps, Docker e Git na construção de uma aplicação simples, segura, executável e documentada, conforme estudo de caso da Aula 1.

## Tecnologias

- **Back-end:** Python + Flask
- **Containerização:** Docker + Docker Compose
- **Versionamento:** Git + GitHub
